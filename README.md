## PacketLens

PacketLens is a learning-first network packet analysis tool built from scratch.
The goal is not to replace Wireshark, but to understand how packet capture,
protocol decoding, and traffic analysis actually work under the hood.

## Goals
- Capture raw network packets
- Decode protocols step by step
- Build custom traffic insights
- Learn networking the right way

## Tech Stack
- Python
- Raw sockets / pcap
- Linux
